# Fleet Nexus Brand Color Palette

This document provides the standard color palette based on the finalized Fleet Nexus logo components. Use these hex codes and guidelines to establish a cohesive and professional website design.

---

## 1. Primary Component Palette

| Component | Description | Approximate Hex Code | Suggested Website Use |
| :--- | :--- | :--- | :--- |
| **Logistics Green** | The energetic color used for the body of the truck and the wheels. Represents growth and efficiency. | `#28D07C` | **Primary CTAs** (e.g., "Schedule a Demo"), Action Buttons, Active states. |
| **Corporate Cobalt** | A rich blue used for the letter "N," the surrounding swoop, and the road body. Represents reliability. | `#1A73E8` | **Links**, Section Headings, Navigation elements, Accent Icons. |
| **Fleet Navy** | The dark, grounding color used for the primary typography ("FLEET NEXUS" and subtext). | `#2C4B64` | **Main Body Text**, Primary Headings, Header/Footer backgrounds. |

## 2. Supporting & Neutral Palette

| Component | Description | Approximate Hex Code | Suggested Website Use |
| :--- | :--- | :--- | :--- |
| **Canvas White** | The clean background color from the logo image. | `#FFFFFF` | **Main Page Backgrounds**, Card backgrounds, Container backgrounds. |
| **Soft Gray** | The subtle gradient color seen towards the outer edges of the frame. | `#F4F6F8` | **Alternating Sections**, Zebra-striping tables, Form fields. |
| **Secondary Road Lines** | The lighter blue shade inside the road separation marks. | `#8AB4F8` | **Form Borders**, Subtle dividers, Hover state accents. |

---

## 3. Recommended Website Layout Hierarchy

To ensure the website remains modern, readable, and perfectly aligned with your brand identity, apply the documented colors using this structured approach:

* **Background Dominance (60%):** Use **Canvas White (`#FFFFFF`)** for the main page real estate to maintain a crisp, clean aesthetic. Use **Soft Gray (`#F4F6F8`)** to visually separate sequential sections on long scrolling pages.
* **Text & Structure (30%):** Set your primary headers (`<h1>`, `<h2>`) and heavy navigation blocks to **Fleet Navy (`#2C4B64`)**. This provides excellent readability and an authoritative look compared to standard black text.
* **Accents & Conversions (10%):** Reserve **Logistics Green (`#28D07C`)** exclusively for your highest-priority conversion triggers (like a "Get Started" or "Sign Up" button). Use **Corporate Cobalt (`#1A73E8`)** for interactive text links and secondary layout iconography.
